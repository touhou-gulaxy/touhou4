# 舰船补给机制
# 在星堡以上等级的恒星基地/主宰/停泊在建有补给设施星球上时可以自动补给，且停驻时不消耗补给品
# 补充耗时floor(4+sqrt(舰船数量))天，同时只能给两支舰队进行补给
# 开启并行补给法令时，可以同时给所有停泊舰队补给，但效率下降
# 补给优先级为：补给低于10%的舰队>补给较多的舰队>其他
# 交战时额外增加300%的补给消耗速度，年度结算
#
# 主宰可以为最多三支舰队补给，耗尽后需要12个月补满
# 星基根据等级可以为最多4-5支舰队补给，且补给模块可以增加上限，耗尽后需要6个月补满
# 星球的补给不会耗尽
# 补给舰可以提供舰队共享补给，共享补给会被优先消耗
# 携带合成模块的可以减少甚至不消耗补给
#
# 补给低于60%时，舰船循环降低10%
# 补给低于50%时，舰船循环降低15%
# 补给低于40%时，舰船循环降低25%，伤害降低16%
# 补给低于30%时，舰船循环降低50%，伤害降低32%
# 补给低于20%时，舰船循环降低75%，伤害闪避降低50%
# 补给低于10%时，舰船循环降低99%，伤害双穿闪避降低75%
# 补给低于5%时，舰船循环降低100%，伤害双穿闪避降低至-99%，射速降低至-90%。
# 补给为0时，舰船将无法打出任何输出。
#
# 舰队共享补给 - 可以为当前所在舰队提供额外补给，优先消耗
